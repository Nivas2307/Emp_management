# 🏢 Employee Management System

An advanced **Employee Management System** built with **React + Node.js + Express + MySQL**.  
This project provides a **secure, modern, and responsive platform** for managing employee data, authentication, and CRUD operations with file uploads.

## 📖 Summary

The Employee Management System enables organizations to:  
- Manage employees’ records securely.  
- Handle **Signup/Login with JWT Authentication**.  
- Upload and display employee profile photos (Multer).  
- Provide **role-based access** and **protected routes** with middleware.  
- Perform full **CRUD operations** (Create, Read, Update, Delete).  

This project is designed with **modern frontend animations (GSAP)** and a **scalable backend (Express + MySQL)**.  
It also uses **Axios** for API integration and **CORS middleware** for cross-origin support.

## ⚡ Tech Stack & Packages

- ⚛️ [React.js (Vite)](https://react.dev/) – UI

## 🚀 Features

### 🔐 Authentication
- Signup with **name, email, password, profile photo upload**.  
- Login with JWT-based authentication.  
- Auto-redirect to dashboard after login.  
- Logout clears token/session.  

### 👤 Employee Management
- **CRUD Operations** – Add, view, update, delete employees.  
- Upload and update **employee profile photo** (Multer).  
- Filter & search employees by department, project, or status.  

### ⚙️ Backend Features
- **CORS Middleware** for cross-origin API access.  
- **Router-based structure** for clean APIs.  
- **JWT Middleware** to protect routes.  
- Centralized **error handling middleware**.  

### 🎨 Frontend Features
- Responsive React frontend (desktop, tablet, mobile).  
- Animated transitions using GSAP.  
- Protected routes with React Router.  
- API calls with Axios for smooth data handling.
